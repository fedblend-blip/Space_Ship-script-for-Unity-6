# Space_Ship-script-for-Unity-6
This is a beginner-friendly spaceship control script. The physics are very simple, so any beginner will understand how it works. Scripts are also provided for controlling cannons (firing) for laser or standard cannons: a cannon control script (handling button presses, etc.) and a bullet script that adds speed to the bullet.
